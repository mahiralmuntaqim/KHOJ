🚨 Feature 1 — Fraud Detection

🚨 Fraud Detection & Flag System

Automatically identifies and flags suspicious activities to prevent misuse. Customers can report issues manually. The system also auto-detects patterns like spam bookings and high cancellation rates.

1

Customer Reports

Customer submits a report about a suspicious listing or user

2

Auto Detection

System checks booking patterns, cancellation rate, and repeat reports

3

Flag Created

MongoDB stores the flag with severity level and details

4

Admin Reviews

Admin resolves or dismisses — user may be banned or cleared

Report a suspicious listing, user, or booking

🔍 Check My Flag Status

See if your account has any active flags

🛡️ Admin — View All Flags

See all open fraud flags in the system
🔑 Requires: admin login token
