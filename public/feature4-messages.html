💬 Feature 4 — In-App Messages

💬 In-App Messaging System

Secure communication between customers and providers. Booking notifications are automatically sent to providers as messages when a booking is created. Both sides can chat in real-time within the platform.

📬

Auto Booking Notifications

When a customer books, a system message is instantly sent to the provider's inbox

💬

Two-Way Chat

Customer and provider can message each other directly, linked to a booking

🔔

Unread Count

Unread badge shown in the navbar — updates every 30 seconds automatically

📤 Send a Message

Send a text message from one user to another
🔑 Requires: sender's JWT token

📥 Get Inbox

All conversations — latest message per person

💬 View Conversation

Full message thread between you and another user
Load a conversation to see messages

📬 Booking Notifications (Provider)

All booking notification messages received
🔑 Requires: provider JWT token
